Features

Big Customization

Choose from multiple themes, solid colors, or even your osu! profile cover.

All Game Modes

Display your stats for any mode: osu!, mania, fruits, or taiko.

Easy Embedding

Get ready-to-use snippets for BBCode, Markdown, and HTML to share anywhere.

Two Card Styles

Pick between a compact 'mini' card or a 'full' card with more details.

About This Project

This is a personal project built with ♥ by me for the osu! community. The goal was to create something like github readme cards but for osu, I started the project in december and added many features I thought would be cool to have in a card, and I'm trying to keep implementations working as the game updates.

It's open source! If you have feedback, find a bug, or want to contribute, please visit the repository.
